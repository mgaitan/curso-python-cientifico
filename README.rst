Parte de este material ha sido inspirado y/o adaptado de múltiples fuentes


* `Scipy Lectures <http://scipy-lectures.github.io/>`_

  de J.R. Johansson

  traducción al castellano por la comunidad `Python Argentina <http://python.org.ar/>`_
